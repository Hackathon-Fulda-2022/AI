# AI

###Description:###

```
A complete voice recognition software
developed for use in nursing stations 
to improve both patients lining conditions
and nurses working conditions.
```

---

###Dataflow:###

```
The voice recognition using whisper is
activated using the phrase "Hallo Anna".
This enables the recording. After the
termination phrase "Danke Anna" the data
is, if necessary, sent to deepl to be
translated into german.The voice recording
is the sent to an interpreter as a single
string to be split up and organized into
a useful format. After that the data is
added to the database to be displayed on
the website and further analysed.
```
